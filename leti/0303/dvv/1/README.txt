Requirements:
1) MSVC 2015
2) Boost 1.60.0
3) libprotobuf 3.0.0

Need to generate the Message.pb.h and Message.pb.cc.

To start server run ChatServer.exe. You will be offered to choose port and number of threads.
To start client run ChatClient.exe. You will be offered to choose host ip, port and nickname.

Commands:
/c stopServer
